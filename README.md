I'm not an engineer by title — I'm someone who gets obsessed with ideas and won't stop until they're alive and working.

I've built AI agents, network tools, live event platforms, encrypted notes apps, repo health scorers, HR systems, e-commerce stores, and a Telegram bot that controls MikroTik routers. Not because I was assigned to — because something clicked and I had to see if it could exist.

The stack changes. The itch doesn't.
