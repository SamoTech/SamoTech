## About

I build practical AI systems and automation tools that solve real problems — not demos, production systems.

Currently focused on:
- **AI agent orchestration** using LangGraph, AutoGen, and CrewAI
- **Full-stack web applications** with Next.js, React, and Node.js
- **Network automation** for MikroTik infrastructure
- **Affiliate & e-commerce** platforms with real-time data pipelines
